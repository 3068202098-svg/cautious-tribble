# C51 单片机 4×4 矩阵计算器（8 位数码管显示）

这个示例使用 **AT89C52 / STC89C52（兼容 8051）**，完成：
- 4×4 矩阵加法
- 4×4 矩阵减法
- 4×4 矩阵乘法
- 通过 8 位数码管动态扫描显示结果

代码文件：`matrix_calc_4x4.c`

## 一、硬件连接（示例）

- 段选：P3.4(P_SER)、P3.5(P_SRCLK)、P3.6(P_RCLK) 接 74HC595
- 位选：P2.0~P2.7 控制 8 位数码管位选（低有效）
- 数码管：共阴极（段码表已按共阴配置）
- 晶振：11.0592MHz（定时器重装值按该频率配置）

> 若你硬件是共阳极，需将段码取反并调整位选极性。

## 二、显示格式

每次显示一个矩阵元素，8 位内容为：

- 第 1 位：行号（1~4）
- 第 2 位：列号（1~4）
- 第 3 位：空白
- 第 4 位：符号位（负数显示 `-`）
- 第 5~8 位：数值（右对齐，最多 4 位）

例如：`2 3 - 0 1 2 8` 表示 `C[2,3] = -128`（行列从 1 开始显示）。

## 三、程序流程

1. 定义矩阵 `A`、`B`
2. 依次执行 `A+B`、`A-B`、`A×B`
3. 将结果存入 `C`
4. 按元素轮询显示 `C[0][0] ... C[3][3]`
